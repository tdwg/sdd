Resource directory for "encoded-inline-formatting" folder

This folder contains a proposal for inline text formatting that may be embedded/encoded in database or xml field content.

Resources

  • The proposal itself: EncodedFormattingProposal.html.
    •
  • Short sample xslt script, data, and transformed output have been created as proof of concept, see:
    • UBIF-EncodedFormatting-Recover.xsl (recover and present and xhtml) and
    • UBIF-EncodedFormatting-Strip.xsl (remove encoded formatting in support of text comparison or indexing).
    • UBIF-EncodedFormatting-SampleData.xml (example data, using a generic format, not bound to any UBIF schema)
    • UBIF-EncodedFormatting-Recover-SampleOutput.html (expected result of "recover"-transformation)
    • UBIF-EncodedFormatting-Strip-SampleOutput.html) (expected result of "strip"-transformation)
  • For further information see also the SDD Home Page (Wiki)!
