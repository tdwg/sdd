Resource directory for "enum-specs" folder

The folder "enum-specs" contains additional material relating to value enumerations (controlled vocabularies) found in the UBIF and SDD schemata. For more information on UBIF and SDD see the SDD Wiki, especially the Charter.

Note: While this document is human-readable, RDDL-aware processors may use it to automatically retrieve resources such as different schema versions for validating documents. See http://www.rddl.org for further information.

Resources

  • To simplify application building, enumerated values present in the UBIF schema are also provided as SDD data files (using an xslt script that converts schema information into ubif:DescriptiveConcept/ConceptStates):
    • UBIF_EnumData.xml and
    • SDD_EnumData.xml.
  • These files contain human readable labels, abbreviations and descriptions in standard UBIF format. In addition, some enumerations provide extended specifications using a separate namespace and specified in a separate schema "UBIF_EnumSpecs.xsd".
  • See also the general documentation RDDL subdirectory, especially:
    • UBIF_EnumData.html
    • SDD_EnumData.html.
